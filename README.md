# TodayTravel
## 개요
  - 네이버맵 API 사용해 만들어본 간단한 도보여행 기록 관련 토이프로젝트
  - 지도를 길게 클릭하면 마커를 생성하여 원하는 목적지 선택 가능
  - 플로팅 버튼 클릭 시 임의 좌표로 마커를 생성하여 목적지 설정 가능
  - 지난 여행 기록을 지도에 표시 가능
  - 여행/지도 관련 사용자 설정 가능
